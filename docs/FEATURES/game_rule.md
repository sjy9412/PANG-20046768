# 게임 룰 (Game Rules)

## 개요

PANG의 핵심 규칙을 정의한다. 모든 미션과 스테이지에 공통 적용된다.

---

## 1. 목표

화면에 등장하는 **모든 버블(Bubble)을 제한 시간 내에 제거**하면 스테이지를 클리어한다.

---

## 2. 버블(Bubble) 규칙

### 크기와 분열

| 크기 | 와이어 피격 시 결과 |
|------|-------------------|
| Large | Medium 2개로 분열 |
| Medium | Small 2개로 분열 |
| Small | 소멸 |

### 이동 패턴
- 버블은 좌우로 이동하며 벽, 바닥, Block 윗면에 부딪히면 반사된다.
- 크기가 작을수록 튀는 높이가 낮고 이동 속도가 빠르다.
- 버블은 플레이어를 통과하지 않으며, 닿으면 플레이어가 데미지를 입는다.

---

## 3. 플레이어 규칙

### 이동
- 좌우 이동만 가능하며, 점프는 불가능하다.

### 공격 — 와이어(Hook)
- 위 방향으로 와이어를 발사한다.
- 와이어는 천장에 닿거나 버블에 닿을 때까지 직선으로 뻗는다.
- **한 번에 하나의 와이어만 화면에 존재**할 수 있다.
- 와이어가 화면에 있는 동안 재발사가 불가능하다.

### 목숨
- 시작 시 목숨 3개 지급.
- 버블에 닿거나 제한 시간이 초과되면 목숨 1개 감소.
- 목숨이 0이 되면 게임 오버.

---

## 4. 장애물(Block) 규칙

- 스테이지에 고정 배치된 구조물이다.
- **파괴 불가**: 와이어나 버블 충돌로 제거되지 않는다.
- 버블은 Block 윗면에서 반사되어 위로 튕겨 오른다.
- 플레이어가 위치한 곳에는 Block이 생성되지 않는다.

---

## 5. 제한 시간

- 각 스테이지마다 제한 시간이 설정된다.
- 타이머는 스테이지 시작과 동시에 감소한다.
- 시간 초과 시 버블 처리 여부와 관계없이 목숨 1개 감소 후 스테이지 재시작.

---

## 6. 아이템(Power-up)

버블을 터뜨릴 때 일정 확률로 아이템이 드롭된다.

| 아이템 | 효과 | 지속 |
|--------|------|------|
| 더블 훅 | 와이어를 2개 동시에 발사 가능 | 일정 시간 |
| 다이너마이트 | 화면의 모든 버블 즉시 제거 | 즉발 |
| 시계 | 버블의 움직임 일시 정지 | 일정 시간 |
| 방어막 | 버블 접촉 무적 | 일정 시간 |
| 보너스 점수 | 음식·보물 등 추가 점수 획득 | 즉발 |

---

## 7. 점수 규칙

| 조건 | 점수 |
|------|------|
| Small 버블 제거 | 100점 |
| Medium 버블 제거 | 200점 |
| Large 버블 제거 | 500점 |
| 스테이지 클리어 보너스 | 남은 시간(초) × 10점 |
| 아이템 획득 | 아이템별 상이 |

---

## 8. 스테이지 클리어 / 게임 오버

- **클리어 조건**: 제한 시간 내 화면의 모든 버블 제거
- **게임 오버 조건**: 목숨이 0이 된 상태에서 추가 피격 또는 시간 초과
