# 메인 화면 (Main Screen)

## 개요

게임 실행 시 가장 먼저 표시되는 화면이다. 플레이어가 게임 시작, 모드 선택 등을 결정하는 진입점 역할을 한다.

---

## 화면 구성 요소

### 1. 타이틀 로고
- 화면 상단 중앙에 **"PANG"** 로고를 표시한다.
- 애니메이션 효과(바운스 또는 페이드인)로 등장한다.

### 2. 메뉴 항목

| 항목 | 설명 |
|------|------|
| 1P GAME | 1인 플레이 모드로 게임 시작 |
| HIGH SCORE | 역대 최고 점수 목록 표시 |

- 방향키(↑ / ↓)로 항목을 선택하고, Enter 또는 Space로 확정한다.
- 현재 선택된 항목은 하이라이트(색상 강조)로 구분된다.

### 3. 배경
- 게임 월드를 연상시키는 레트로 배경 이미지 또는 애니메이션을 표시한다.
- 버블이 화면을 가로질러 튀는 데모 애니메이션을 보여줄 수 있다.

### 4. 조작 안내
- 화면 하단에 간단한 키 조작 안내를 표시한다.
  - 예: `← → 이동 / SPACE 발사 / ENTER 선택`

---

## 화면 전환

| 조건 | 전환 대상 |
|------|-----------|
| 1P GAME 선택 | 게임 플레이 화면 (Stage 1 시작) |
| HIGH SCORE 선택 | 최고 점수 목록 화면 |
| 일정 시간 방치 | 데모 플레이(어트랙트 모드) 자동 재생 |
